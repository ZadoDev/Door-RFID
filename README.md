# Door-RFID
simple IOT project that can to open the door by using RFID and mobile app (Development)
# Microcontroller
- Arduino Nano
- Esp01
# Modules
- RFID-RC522
- LCD
- Relay
- BUTTON 4 number (up,down,enter,exit)
- Buzzer
- switch button
